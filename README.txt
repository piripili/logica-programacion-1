# Ejercicio de Lógica de Programación 1

Este proyecto corresponde a la práctica de **ordenamiento de números en JavaScript**.  
El programa solicita tres números al usuario, los analiza y muestra los resultados ordenados de mayor a menor y de menor a mayor. También identifica si los números son iguales.

---

## Requerimientos
- Solicitar al usuario 3 números mediante `prompt`.
- Convertir las entradas a tipo `Number` para evitar errores.
- Validar que las entradas sean números.
- Identificar el número mayor, el del centro y el menor.
- Mostrar los números ordenados:
  - De mayor a menor.
  - De menor a mayor.
- Detectar si los números son iguales y mostrar un mensaje.
- Mostrar resultados en consola y en el DOM.

---

## Tecnologías utilizadas
- **HTML5** para la estructura básica.
- **JavaScript** para la lógica del programa.
